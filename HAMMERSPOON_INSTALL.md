# Hammerspoon インストールガイド

## 📥 ステップ1: Hammerspoonをダウンロード・インストール

### 方法A: 公式サイトからダウンロード（推奨）

1. ブラウザで https://www.hammerspoon.org にアクセス
2. **Download** ボタンをクリック
3. ダウンロードした `.dmg` ファイルを開く
4. **Hammerspoon.app** を **Applications** フォルダにドラッグ
5. アプリケーションフォルダから **Hammerspoon** を起動

### 方法B: Homebrewでインストール

もしHomebrewがインストール済みなら：

```bash
brew install --cask hammerspoon
```

## 🔐 ステップ2: 権限設定

初回起動時に以下の設定が必要です：

1. Hammerspoonを起動すると、メニューバーにアイコンが表示されます
2. **アクセシビリティ権限**を求められるので「許可」
3. **システム設定** → **プライバシーとセキュリティ** → **アクセシビリティ**
4. **Hammerspoon** がリストにあり、オンになっていることを確認

## ✅ 確認

- メニューバーにHammerspoonのアイコン（🔨マーク）が表示されていればOK
- アイコンをクリック → **Console** を選択してエラーがないか確認

インストールが完了したら、次のステップに進みます！
